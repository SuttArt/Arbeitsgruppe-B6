Willkommen beim ASCII Generator!
Diese Webseite wandelt jedes Zeichen einer Zeichenkette in das entsprechende dezimale, oder aber auch hexadezimale Zeichen um. Zudem können Sie hier weitere Informationen zum Begriff ASCII als auch deren Grundlage für weitere Codierungen. Die ASCII, oder aber auch ausgeschrieben "American Standard Code for Information Interchange", ist eine 7-Bit Zeichenkodierung, welche als Grundlage für spätere mehr Bit basierende Kodierungen gilt.
Mithilfe der jeweiligen ASCII Tabelle kann ein jeweiliges Zeichen in seinen entsprechenden Wert umgewandelt werden. Im Laufe der Zeit wurden weitere Codierungsformen etabliert - so unter anderem der Codierungsstandart "UTF8".

Kodierung mithilfe von ASCII

Jedes Zeichen, welches durch ASCII dargestellt werden kann wird je einem Muster aus 7-Bit zugeordnet. Da jedes Bit 2 Zustände annehmen kann, folgt somit dass es 27, somit 128 verschiedene Bitmuster zu Verfügung stehen. Die Menge der durch ASCII darstellbaren Mengen wären somit 128 verschiedene Zeichen, wovon wiederum 95 druckbare Zeichen sind und die restlichen 33 Zeichen nicht druckbare Zeichen sind. Diese sind alle einem jeweiligem Zahlenwert zugeordnet. Diese Zahlwerte können entweder als Dezimalzahl, Hexadezimalzahl oder aber auch als Oktadezimalzahl dargestellt werden.

kurze Geschichte zu ASCII

Dieser verdrängte ab 2008 den ASCII-Code, da der UTF8-Code einerseits abwärtskompatibel ist und andererseits eine weitere Zeichenmenge beinhaltet. Dennoch hat der ASCII-Code selbst heutzutage noch Anwendungsgebiete. So wird dieser unter anderem noch beim Erstellen von E-Mail oder bei der Domainauflösung mit Umlauten verwendet. Auf jeden Fall wird der Text in beiden Fällen vorm Übertragen in das ASCII-Format konvertiert. Das geschieht jedoch meist vollkommen automatisch.

  • ASCII-Dezimal Wert
  • ASCII-Hexadezimal Wert
  • ASCII-Oktadezimal Wert
  • Zeichen

Sollte man jedoch einen formatierten Wert haben und man möchte diesen in sein entsprechendes Zeichen umwandlen, so kann man im Internet speziell sortierte Tabelle finden, welche das Übersetzen leichter machen.

Eine ASCII Tabelle sieht unter anderem folgendermaßen aus:

Dec Hex Oct Char
0 0 000 NUL (null)
1 1 001 SOH (start of heading)
2 2 002 STX (start of text)
3 3 003 ETX (end of text)
